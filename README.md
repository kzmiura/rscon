# rscon

Source RCON Client in Rust.


# Usage

```bash
rscon --help
```

```
Usage: rscon --address <ADDRESS> --password <PASSWORD>

Options:
  -a, --address <ADDRESS>    
  -p, --password <PASSWORD>  
  -h, --help                 Print help
  -V, --version              Print version
```

# build

```bash
cargo build
```
